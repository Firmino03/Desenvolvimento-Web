Resumo do Projeto

Este site é um projeto para a disciplina de Desenvolvimento Web do 2º período de Informática para Internet.

Durante a criação do site, enfrentamos desafios como estruturação correta em HTML, organização de conteúdo, aplicação de estilos em CSS e uso de navegação entre páginas.

Os principais pontos utilizados foram: semântica HTML5, listas, tabelas, links internos/externos e imagens.
